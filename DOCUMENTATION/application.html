Base de Données

La Base de données est à créer manuellement.

Nous utilisons un SGBDR SQL.

Afin de créer cette base, il faut utiliser le fichier /SQL/mediabase.sql

Par ailleurs, les seuls visiteurs ayant la capacité de déposer des médias
ont un pseudo et un mot de passe.

Il faut également créer manuellement dans la table "users" le visiteur
en choisissant ses identifiants.

Système Applicatif

Le Langage de Programmation est PHP

La conception nécessite l'utilisation d'un navigateur html5

L'Application est optimisée pour les Smartphones

L'Application débute avec la page "index.php"

L'Application est structuré en deux types de programmes:

D'une part: les classes, accessibles par un système "autoload" appelé par l'index

D'autre part: les objets, permettant ainsi d'interagir à la demande du visiteur.
